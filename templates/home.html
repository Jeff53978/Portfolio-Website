    Microcord is de enigste Discord API wrapper voor Microcontrollers. Aangezien er geen websocket modules zijn voor micropython heb ik de RFC 6455 websocket protocol gereverse engineered, en gemaakt met de ingebouwde socket module. De chip verbindt via de eigen websocket met de discord gateway en om ram te besparen wordt de READY opcode van de gateway niet opgeslagen, een minimale discord bot kan functioneel draaien op 64kb ram! Dat is ~300x zo efficient als een gemiddelde discord api wrapper. Deze module wordt ondersteund op de RP2040 en ESP32 microcontrollers.

    Xeonpanel is een een paneel gemaakt met Flask waarop je game-servers kan hosten maar ook nodejs of python bots, dit project bestaat uit 2 onderdelen, het paneel en deamon.


    Het paneel is de website en backend waarop je alles ziet en kan beheren.

    De deamon is een python script dat draait op een node (vps of dedicated server) zodat het paneel ermee kan verbinden en er servers op kan aanmaken via Docker.
